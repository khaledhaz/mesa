# Mesa — pay-at-table, perfected
A QR scan-to-pay SaaS for restaurants. Diners scan, split fairly (claim your own items),
tip kindly, and pay in seconds — no app, no login. Restaurants get a real-time floor board,
revenue analytics, built-in loyalty, transparent flat pricing, and 60-second setup.

- Landing: `index.html`
- Diner demo: `pay.html?code=OE-03` (also `?code=OE-05`)
- Restaurant dashboard: `dashboard.html` (demo login button)

Backend: Supabase (Postgres + Auth + RLS + Realtime). Payments are simulated in this build.
